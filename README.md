# 🚀 Personal Portfolio

![Portfolio Screenshot](https://sprightly-syrniki-2a2ab3.netlify.app)

My personal portfolio website built with React, Vite, and Framer Motion. Features an AI avatar video introduction, interactive UI, and a modern dark theme design. Showcases my projects, skills, certifications, and experience as a Full Stack Developer.

## 🌐 Live Demo

[View Portfolio](https://sprightly-syrniki-2a2ab3.netlify.app)

## ✨ Features

- 🎥 **AI Avatar Video Introduction** - Automatic video with click to unmute
- 🎨 **Modern Dark Theme** - Clean, professional design with red accent
- 📱 **Fully Responsive** - Works on all devices
- 🎞️ **Smooth Animations** - Framer Motion for scroll-triggered effects
- 📊 **Proficiency Bars** - Visual representation of skills
- 🏆 **Certifications** - Click to view certificate images
- 📬 **Contact Form** - Interactive contact section with working links
- 🚀 **Projects Showcase** - Live demo and GitHub links

## 🛠️ Tech Stack

| Technology | Purpose |
|------------|---------|

| **Vite** | Build Tool |

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/samriddhi7777/Portfolio.git

# Navigate to project
cd Portfolio

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react) uses [Oxc](https://oxc.rs)
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react-swc) uses [SWC](https://swc.rs/)

## React Compiler

The React Compiler is not enabled on this template because of its impact on dev & build performances. To add it, see [this documentation](https://react.dev/learn/react-compiler/installation).
